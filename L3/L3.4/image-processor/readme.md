# Image Processor

## 🚀 Запуск проекта

### 1. Подготовка

```bash
cp .env.example .env
```

### 2. Запуск (Docker)

```bash
make up
```

### 3. Остановка

```bash
make down
```

---

## 🛠 Локальный запуск

### Запуск API

```bash
make run-api
```

### Запуск worker

```bash
make run-worker
```

---

## 🧪 Тесты

```bash
make test
```

---

## 🗄 Миграции

```bash
make migrate
```

---

## 📦 Сборка

```bash
make build
```
